## Prerequisites

- Node.js 16.X
- npm 8.X

## Local setup

```bash
# checkout repo
git clone https://github.com/bosonprotocol/widgets.git

# install deps
cd widgets
npm ci
```

## Building

```bash
npm run build
```

## Running locally

```bash
npm run dev
```
